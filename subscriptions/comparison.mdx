---
title: Comparison
description: 'Side-by-side comparison of Quark subscription tiers'
---

| Feature               | Free               | Quark Pro Lite     | Quark Pro          | Quark Pro Ultra   |
| --------------------- | ------------------ | ------------------ | ------------------ | ----------------- |
| Log older messages    | Up to 12 hours old | Up to 12 hours old | Up to 8 weeks old* | Up to 8 weeks old |
| Change log format     | No                 | Yes                | Yes                | Yes               |
| Customize colors      | No                 | Yes                | Yes                | Yes               |
| Per log type channels | No                 | Yes                | Yes                | Yes               |
| Log bot actions       | No                 | Yes                | Yes                | Yes               |
| Quark Pro bot         | No                 | No                 | Yes                | Yes               |
| Custom bot            | No                 | No                 | No                 | Yes               |

<Note>*Quark Pro initially allows messages up to 2 weeks old to be logged, however this increases to 8 weeks
if the subscription is active for 6 months.</Note>
